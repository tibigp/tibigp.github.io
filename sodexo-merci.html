Sodexo Merci

The request: to build the frontend based on the design provided by the client, build the header animations and build a newsletter.

View website

Header animation

The header animation is structured in 2 parts. First I've made the fireworks animation in Adobe After Effects and I've export it as a video file. And the second animation I've made it in JavaScript and it is initiated by the ending of the fireworks video. The number displayed in the animation is taken from the URL.

Newsletters

I've build the newsletter using HTML tables for the content structure to support older email clients. The newsletter is responsive. It was tested in Litmus for more than 25 email clients. The link from the newsletter contains a custom parameter with a unique numeric value for each recipient. The value is displayed in the animation from the header of the site.
